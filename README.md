# AlgorithmicFoundation

A complete collection of fundamental algorithms implemented in Java.

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## Searching Algorithms
- Linear Search
- Binary Search
- Ternary Search
- Exponential Search
- Jump Search

## Graph Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Prim's Algorithm
- Topological Sort
- Dijkstra's Algorithm
